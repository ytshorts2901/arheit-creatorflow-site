Data deletion

Users can disconnect TikTok and delete local application data at any time.

  1. Revoke Arheit CreatorFlow in TikTok’s connected-app settings.
  2. Use the application’s disconnect function to delete locally stored OAuth tokens.
  3. Delete local upload logs and selected video files from the device if no longer needed.

For assistance, email ytshorts2901@proton.me. Do not include passwords or tokens.
