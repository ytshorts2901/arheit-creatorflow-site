Before contacting support

  • Do not send passwords, client secrets, access tokens, or refresh tokens.
  • Include the application version and a non-sensitive error message or TikTok log ID.
  • State whether the issue concerns login, file validation, draft upload, or inbox notification.

Disconnecting TikTok

You may revoke the application’s access through TikTok account settings and remove locally stored credentials from the application.
