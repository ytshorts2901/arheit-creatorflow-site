Prepare videos. Send drafts. Review before posting.

Arheit CreatorFlow is a desktop application that helps authorized creators validate local short-form videos and send them to their own TikTok inbox as drafts. The creator completes editing and publishes manually in TikTok.

Secure authorization

Creators connect their own TikTok account through TikTok Login Kit using OAuth 2.0 with PKCE.

Local validation

The application checks file format, duration, resolution, size, and integrity before upload.

Draft-only workflow

Videos are sent to the authorized creator’s TikTok inbox as drafts. The application does not use Direct Post.

Manual review

The creator opens the TikTok inbox notification, reviews the draft, adds platform features, and publishes manually.

How it works

  1. The creator opens the desktop application.
  2. The creator chooses “Connect TikTok” and authorizes the requested scopes.
  3. The creator selects a local MP4/H.264 video.
  4. The application validates the file and asks for confirmation.
  5. The video is uploaded to TikTok as an inbox draft through the official Content Posting API.
  6. The creator opens TikTok, reviews the draft, and decides whether to publish it.
Arheit CreatorFlow does not automate likes, comments, follows, views, or other engagement. It does not scrape TikTok and does not publish without the creator’s action in TikTok.

Security and privacy

OAuth tokens are stored securely for the authorized creator and can be revoked. Local files remain on the creator’s device unless the creator explicitly starts a draft upload. Read the Privacy Policy for details.
